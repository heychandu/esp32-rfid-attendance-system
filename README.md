## 📌 Overview
This project is an ESP32-based RFID attendance system built step-by-step.

## 🟢 Current Version

- Displays output in Serial Monitor
- Uses SPI communication

## 🧠 Technologies Used

- MFRC522 RFID Module
- SPI Communication
- Arduino IDE
