# 📡 ESP32 RFID Attendance System

> Developed by **Chandu R**

---

## 📌 Overview
This project is a **smart RFID-based attendance system** built using ESP32.  
It evolves step-by-step from a basic RFID reader into a complete system with user interface, feedback, and attendance logic.

---

## 🚀 Features

- 📡 RFID Card Detection  
- 📟 LCD Display Interface  
- 🔊 Buzzer Feedback  
- 💡 LED Indication  
- 🔁 IN / OUT Attendance Logic  
- 🧠 UID-based User Identification  
- ⚙️ Modular & Scalable Design  

---

## 🧠 System Workflow

👉 Scan Card → 🔍 Read UID → 👤 Identify User → 📊 Mark Attendance → 📟 Display Result → 🔊 Feedback  

---

## 🧩 Hardware Used

- ESP32  
- MFRC522 RFID Module  
- I2C LCD (16x2)  
- Buzzer  
- LED  

---

## ⚙️ Technologies Used

- Embedded C/C++ (Arduino IDE)  
- SPI Communication (RFID)  
- I2C Communication (LCD)
